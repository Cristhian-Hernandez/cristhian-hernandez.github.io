Introducción

¿Que es esto y cual es su propósito?

Esta pagina web fue creada con el propósito de facilitar la entrega de tareas y como tarea final del ciclo 5 (décimo grado) en el colegio Sandalphon del grupo educativo Bacatá.  Más en especifico, esta pagina pertenece a Cristhian Hernández del grupo C5A.

 

Para lograr el diseño y funcionalidad de este sitio web, se han utilizado una serie de tecnologías. Estas son HTML 3.0, CSS, Bootstrap y el lenguaje de programación JavaScript y algunas de sus bibliotecas disponibles, principalmente jQuery.

 

Anexo vas a encontrar las tareas asignadas por el profesor Jaime Gallego. Cada una con su breve descripción y un archivo incluido.

 

Para navegar por esta pagina web puedes usar los botones hallados en la parte superior izquierda.
